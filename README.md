# GameTheoryViz
Interactive Visualization of Game Theory Concepts | CSE 631 course project


## Installation
```bash
# -- windows --
# 0. Clone repository
# 1. setup venv
python -m venv .venv
# 2. run venv
.venv\scripts\activate
# 3. install requirements
python -m pip install -r requirements.txt
# 4. run streamlit
python -m streamlit run app.py
```
